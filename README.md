Install Python

https://www.python.org/downloads/


Install iD01t App Dependencies


Open Terminal (Utilities)


Type:

cd (drag and drop Installation folder)

Press Enter

Type:

bash install_deps.sh

Press Enter

Type:

cd..

Press Enter

Type:

python3 iD01t_SAM_Reader.py

Press Enter
